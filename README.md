# cobra sample `sample-cli`

A sample cli written in Golang with [cobra](https://github.com/spf13/cobra), which is commonly used by go-based cli tools.

## Install

```
go install github.com/nakamasato/cobra-sample/sample-cli
```

## Usage

```
sample-cli
Hello world!
```
